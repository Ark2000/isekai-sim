# 水文循环

这一阶段开始上强度了

- [ ] 水的模拟，你用笔刷放一滩水，能够自动满溢，水往低处流，形成海洋湖泊之类的，需要保证水的总量是守恒的！

⭐好玩！结合地形绘制，你可以放置水源，看你刷出来的水填满地图的低洼处，你还可以自己挖一条河出来！水的动态性很强！很有观赏性。

- [ ] 开始有温度，温度只和地形挂钩，高的地方冷，低的地方暖和，然后温差会形成风。

- [ ] 水会蒸发，变成水蒸气和云，云会结成团移动，云会消散，会降水，重返地面。
